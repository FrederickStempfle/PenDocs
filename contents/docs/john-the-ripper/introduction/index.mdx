Welcome to the comprehensive John the Ripper guide. This documentation provides detailed instructions on using John the Ripper, one of the most powerful and versatile password cracking tools available for penetration testing and security auditing.

## Getting Started with John the Ripper

<Stepper>
  <StepperItem title="Install John the Ripper">
    <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
      <div>
        <h4 className="font-semibold">On Kali Linux (Pre-installed)</h4>
        <p className="text-sm">John the Ripper comes pre-installed on Kali Linux. Verify with:</p>
        ```bash
        john --version
        ```
        
        <h4 className="font-semibold mt-4">Update to Latest Version</h4>
        <p className="text-sm">To get the latest version:</p>
        ```bash
        sudo apt update
        sudo apt install john
        ```
      </div>
      
      <div>
        <h4 className="font-semibold">From Source (Jumbo Version)</h4>
        <p className="text-sm">For the feature-rich Jumbo version:</p>
        ```bash
        # Install dependencies
        sudo apt install build-essential libssl-dev
        
        # Clone repository
        git clone https://github.com/openwall/john -b bleeding-jumbo john
        
        # Build from source
        cd john/src
        ./configure && make -s clean && make -sj4
        ```
        <p className="text-sm mt-2">The compiled binary will be in the `john/run` directory.</p>
      </div>
    </div>
  </StepperItem>
  
  <StepperItem title="Verify Installation">
    <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
      <div>
        <h4 className="font-semibold">Check Version and Features</h4>
        <p className="text-sm">Verify your installation:</p>
        ```bash
        john --version
        ```
        
        <p className="text-sm mt-2">List supported hash formats:</p>
        ```bash
        john --list=formats
        ```
      </div>
      
      <div>
        <h4 className="font-semibold">Test with Sample Hash</h4>
        <p className="text-sm">Create a test file with a simple hash:</p>
        ```bash
        echo "user:AZl.zWwxIh15Q" > test.txt
        
        # Try to crack it
        john test.txt
        
        # Show cracked password
        john --show test.txt
        ```
        <p className="text-sm mt-2">The password should be "password".</p>
      </div>
    </div>
  </StepperItem>
  
  <StepperItem title="Basic Usage">
    <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
      <div>
        <h4 className="font-semibold">Crack a Password File</h4>
        <p className="text-sm">Basic syntax for cracking:</p>
        ```bash
        # Auto-detect hash type and crack
        john hashfile.txt
        
        # Use a specific wordlist
        john --wordlist=/path/to/wordlist.txt hashfile.txt
        
        # Specify hash format
        john --format=raw-md5 hashfile.txt
        ```
      </div>
      
      <div>
        <h4 className="font-semibold">View Results</h4>
        <p className="text-sm">Display cracked passwords:</p>
        ```bash
        # Show cracked passwords
        john --show hashfile.txt
        
        # Show cracked passwords with statistics
        john --show --stats hashfile.txt
        
        # Show password count only
        john --show --count hashfile.txt
        ```
      </div>
    </div>
  </StepperItem>
</Stepper>

Each section provides detailed explanations, command examples, and real-world use cases to help you master John the Ripper for ethical penetration testing.

<Note title="Ethical Usage" type="warning">
John the Ripper is a powerful tool that should only be used for legitimate security testing with proper authorization. Unauthorized testing of systems you don't own is illegal and unethical. Always obtain explicit written permission before conducting any password cracking activities.
</Note>

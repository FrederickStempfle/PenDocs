# Password Cracking Techniques

This section covers the various techniques available in John the Ripper for cracking passwords, from basic dictionary attacks to advanced rule-based approaches.

        <h3 className="font-semibold">Brute Force Attacks</h3>
        <div className="bg-muted p-3 rounded-md mt-2">
          <h4 className="font-bold">What is a Brute Force Attack?</h4>
          <p className="text-sm">A brute force attack systematically tries all possible character combinations until the correct password is found. John the Ripper implements this through its "incremental" mode.</p>
          
          <h4 className="font-bold mt-3">Basic Syntax</h4>
          ```bash
          # Full brute force (all character sets)
          john --incremental=all hashes.txt
          
          # Digits only (for PINs)
          john --incremental=digits hashes.txt
          
          # Lowercase letters only
          john --incremental=alpha hashes.txt
          
          # Alphanumeric characters
          john --incremental=alnum hashes.txt
          ```
        </div>
        
        <div className="border p-3 rounded-md mt-4">
          <h4 className="font-bold">Incremental Mode Configuration</h4>
          <p className="text-sm">The incremental mode is configured in john.conf:</p>
          ```
          [Incremental:All]
          File = $JOHN/all.chr
          MinLen = 0
          MaxLen = 8
          CharCount = 95
          ```
          <p className="text-sm mt-2">You can create custom incremental modes by adding new sections to john.conf.</p>
        </div>
      </div>

## Hybrid Approaches

<div className="grid grid-cols-1 md:grid-cols-2 gap-4 mt-4">
  <div>
    <h3 className="font-semibold">Combining Techniques</h3>
    <div className="bg-muted p-3 rounded-md mt-2">
      <h4 className="font-bold">Wordlist + Rules + Masks</h4>
      <p className="text-sm">For maximum effectiveness, combine multiple techniques:</p>
      
      <ol className="text-sm list-decimal pl-5 mt-2 space-y-2">
        <li>Generate a base wordlist with target-specific terms</li>
        <li>Apply rules to create variations</li>
        <li>Use mask attacks for specific patterns</li>
        <li>Combine outputs for comprehensive coverage</li>
      </ol>
    </div>
    
    <div className="border p-3 rounded-md mt-4">
      <h4 className="font-bold">Prince Attack</h4>
      <p className="text-sm">Prince (PRobability INfinite Chained Elements) combines words from a wordlist:</p>
      ```bash
      # Generate prince wordlist
      pp64.bin --pw-min=8 --pw-max=16 -o prince_wordlist.txt wordlist.txt
      
      # Use with John
      john --wordlist=prince_wordlist.txt hashes.txt
      ```
    </div>
  </div>
  
  <div>
    <h3 className="font-semibold">Markov Chain Attacks</h3>
    <div className="bg-muted p-3 rounded-md mt-2">
      <h4 className="font-bold">What are Markov Chains?</h4>
      <p className="text-sm">Markov chain attacks use statistical models to generate password candidates based on the probability of character sequences in real passwords.</p>
      
      <p className="text-sm mt-2">This approach is more efficient than pure brute force but more comprehensive than dictionary attacks.</p>
    </div>
    
    <div className="border p-3 rounded-md mt-4">
      <h4 className="font-bold">Using Markov Mode</h4>
      <p className="text-sm">Generate Markov statistics from a wordlist:</p>
      ```bash
      john --markov --mkv-stats=stats.txt wordlist.txt
      ```
      
      <p className="text-sm mt-2">Use Markov mode for cracking:</p>
      ```bash
      john --markov=100:0:1:10 --mkv-stats=stats.txt hashes.txt
      ```
      
      <p className="text-sm mt-2">Where:</p>
      <ul className="text-sm list-disc pl-5">
        <li><code>100</code> is the level (lower = faster but less coverage)</li>
        <li><code>0:1:10</code> is min:max:length (passwords of length 1-10)</li>
      </ul>
    </div>
  </div>
</div>

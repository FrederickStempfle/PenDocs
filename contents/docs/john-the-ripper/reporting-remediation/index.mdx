This section covers how to effectively document password cracking findings and provide actionable remediation recommendations as part of a penetration testing engagement.

        <div className="border p-3 rounded-md mt-4">
          <h4 className="font-bold">NIST SP 800-63B Alignment</h4>
          <p className="text-sm">Modern password guidance from NIST includes:</p>
          <ul className="text-sm list-disc pl-5">
            <li>Minimum of 8 characters (recommend 12+)</li>
            <li>Maximum length of at least 64 characters</li>
            <li>Allow all ASCII and Unicode characters</li>
            <li>Check against lists of commonly used/compromised passwords</li>
            <li>No periodic password changes without reason</li>
            <li>No password hints or knowledge-based security questions</li>
          </ul>
        </div>
      </div>
      
      <div>
        <h3 className="font-semibold">Technical Controls</h3>
        <div className="bg-muted p-3 rounded-md mt-2">
          <h4 className="font-bold">Implementation Recommendations</h4>
          <p className="text-sm">Recommend these technical controls:</p>
          <ul className="text-sm list-disc pl-5">
            <li><strong>Modern Hash Algorithms</strong>: Use bcrypt, Argon2, or PBKDF2</li>
            <li><strong>Salt Implementation</strong>: Unique salt for each password</li>
            <li><strong>High Work Factors</strong>: Increase computational cost</li>
            <li><strong>Password Blacklists</strong>: Block common/compromised passwords</li>
            <li><strong>Multi-Factor Authentication</strong>: Implement where possible</li>
            <li><strong>Password Managers</strong>: Encourage organizational use</li>
          </ul>
        </div>
        
        <div className="border-l-4 border-yellow-500 pl-4 py-2 bg-yellow-50 dark:bg-yellow-900/20 mt-4">
          <h4 className="font-bold">Implementation Example</h4>
          <p className="text-sm">Example bcrypt implementation in Python:</p>
          ```python
          import bcrypt
          
          # Generate a salt and hash a password
          def hash_password(password):
              # Generate a salt with work factor 12
              salt = bcrypt.gensalt(rounds=12)
              # Hash the password
              hashed = bcrypt.hashpw(password.encode(), salt)
              return hashed
          
          # Verify a password against a stored hash
          def verify_password(password, stored_hash):
              return bcrypt.checkpw(password.encode(), stored_hash)
          ```

<Note title="Ethical Reporting" type="warning">
  <p>When reporting password cracking results, follow these ethical guidelines:</p>
  
  <ul className="list-disc pl-5 mt-1">
    <li><strong>Never</strong> include actual cracked passwords in reports or presentations</li>
    <li><strong>Never</strong> share password hashes or cracked passwords with unauthorized individuals</li>
    <li><strong>Always</strong> securely delete password hashes and cracked passwords after testing</li>
    <li><strong>Always</strong> report findings directly to authorized stakeholders through secure channels</li>
    <li><strong>Always</strong> provide clear, actionable remediation steps with your findings</li>
  </ul>
  
  <p className="mt-2">Remember that the goal of password testing is to improve security, not to expose individuals or create additional vulnerabilities.</p>
</Note>
